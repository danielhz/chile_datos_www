# Usar identificadores

<p property="dc:description">Se debe usar un vocabulario preciso para
los elementos referidos en las publicaciones, con la finalidad de
poder distiguir de manera automática cuando nos referimos, en más de
una ocasión, a un mismo elemento y minimizar las situaciones en las
que se produce la ambigüedad. Decimos que la referencia a un objeto es
ambigua cuando hay más de un candidato a ser interpretado como el
referido.</p>
